üìö Understanding Box Styling Properties

üì¶ Padding

What it does: Creates space INSIDE the box, between content and border.

When to use: Give content breathing room, make buttons/cards more comfortable.

Example: padding: 20px; or padding: 10px 20px; (top/bottom left/right)

üéÄ Border

What it does: Creates a visible edge around the box.

When to use: Define boundaries, create separation, add visual interest.

Example: border: 2px solid #3498db; (width style color)

üìè Margin

What it does: Creates space OUTSIDE the box, between this element and others.

When to use: Separate elements, create whitespace, control layout spacing.

Example: margin: 20px; or margin: 0 auto; (center horizontally)

‚≠ï Border Radius

What it does: Rounds the corners of the box.

When to use: Modern look, soften sharp edges, create circles/pills.

Example: border-radius: 8px; or border-radius: 50%; (circle)

üåë Box Shadow

What it does: Adds shadow effects to create depth and elevation.

When to use: Make elements appear floating, create hierarchy, add visual interest.

Example: box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2); (x y blur spread color)

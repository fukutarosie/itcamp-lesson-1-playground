üí° Key Insights

  • Left side (Code): Raw HTML - what developers write
  • Right side (UI): Rendered result - what users see
  • CSS Selectors target the code: But affect the visual UI
  • p { color: blue; } would make all 6 paragraphs blue in the UI
  • .project-card { } would style both project cards
  • #main-header { } would only style the top purple section

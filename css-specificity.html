üéØ What is CSS Specificity?

  • The Problem: Sometimes multiple CSS rules try to style the same element
  • The Solution: The browser uses "specificity" to decide which rule wins
  • The Rule: More specific selectors override less specific ones
  • Think of it as: A "power level" for each CSS rule - highest power wins!

üìä How Specificity is Calculated

Think of specificity as a 4-digit number: [Inline] [IDs] [Classes] [Elements]
Each selector adds points to these categories. Higher total score = Winner! üèÜ

Selector Type Example Specificity Score Explanation
Inline Style style="color: red" 1, 0, 0, 0 Highest power! Beats everything except !important
ID Selector #header 0, 1, 0, 0 Very specific - targets one unique element
Class Selector .button 0, 0, 1, 0 Medium power - can be reused on multiple elements
Element Selector div 0, 0, 0, 1 Lowest power - affects all elements of that type
Multiple Classes .button.primary 0, 0, 2, 0 Each class adds to the score!
Combination #nav .menu li 0, 1, 1, 1 1 ID + 1 class + 1 element
